## 🚀 How to Use This Repository

```bash
# 1. Clone the repository
git clone https://github.com/GajendraAwasthi/Pokhara-University-BCA-All-Semester-Notes-Codes.git

# 2. Navigate to your semester
cd Pokhara-University-BCA-All-Semester-Notes-Codes/Semester-3/

# 3. Open any subject folder and start studying 🎓
```

> 💡 You can also **browse directly on GitHub** — no download needed. Use the search bar to find any subject by name or course code (e.g., `CMP 226`, `Java`, `DBMS`).
